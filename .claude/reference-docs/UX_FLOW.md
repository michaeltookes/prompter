# UX Flow

## Creating a Deck
1. Open Deck Editor
2. Add new card
3. Select layout
4. Enter title, notes, and drop images
5. Repeat for all speaking points

---

## Presenting
1. Open overlay from menu
2. Position near webcam
3. Enable Protected Mode
4. Enable Click-through
5. Use hotkeys to advance cards

---

## During Presentation
- Cmd+Shift+→ next card
- Cmd+Shift+← previous card
- Cmd+Shift+O hide/show overlay
- Cmd+Shift+C toggle click-through

---

## Visual Goal
Presenter never scrolls.  
Presenter only taps next/prev like slides.  
Overlay remains calm, legible, and elegant.
