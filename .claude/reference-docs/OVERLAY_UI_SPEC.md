# Overlay Window UI Spec

## Appearance
- Floating frosted-glass panel
- Soft glow around edges
- Rounded corners
- Minimal UI chrome

Opacity adjustable from subtle to solid.

---

## Card Rendering

Overlay shows ONE card at a time.

Footer (small, auto-hide):
Card 3 / 12
Protected Mode Icon
Click-through Icon

---

### Layout 1: Title + Notes
Large title at top  
Notes below, scrolls if long

### Layout 2: Image Top + Notes
Image top (max 40% height)  
Notes below

### Layout 3: Two Images + Notes
Two equal images  
Notes below

### Layout 4: Grid + Caption
2x2 image grid  
Caption below

### Layout 5: Full Image + Bullets
Image background  
Text over image with subtle dark blur plate behind text

---

## Interaction

Hovering near bottom reveals control strip:
A– / A+ font size
Opacity slider
Toggle click-through

---

## Animation

Card transitions:
- Gentle fade + slide
- 150–200ms duration
- Smooth ease-in-out
